# Дижитал харилцааны ёс зүй

Цахим орчинд харилцахдаа **“дэлгэцийн цаана бодит хүн байгаа”** гэдгийг ухамсарлах нь дижитал соёлын суурь юм. Таны онлайн орчинд үзүүлж буй үйлдэл, харилцаа бүр ШУТИС-ийн академик нэр хүнд болон таны хувийн үнэт зүйлийн илэрхийлэл болно.

---

## Дижитал орчинд баримтлах үндсэн зарчмууд

*ШУТИС-ийн хамт олон, багш, оюутнуудын харилцаанд мөрдөх үндсэн зарчмууд:*

| Зарчим | Агуулга |
| :--- | :--- |
| **1. Хүнлэг байх** | Нүүр тулан хэлж чадахгүй зүйлийг дижитал орчинд бүү бич. Бусдын үзэл бодолд хүлээцтэй, хүндэтгэлтэй хандах. |
| **2. Оюуны өмчийг дээдлэх** | Бусдын бүтээл, санааг ашиглахдаа заавал зохих журмын дагуу иш татаж академик шударга байдлыг эрхэмлэх. |
| **3. Хувийн орон зай** | Бусдын зөвшөөрөлгүйгээр зураг, видео, хувийн мэдээллийг нийтэд дэлгэхээс зайлсхийх. Нууцлалыг эрхэмлэх. |
| **4. Шүүмжлэлт сэтгэлгээ** | Мэдээллийг түгээхээсээ өмнө: "Энэ үнэн үү? Энэ хэрэгтэй юу? Энэ бусдад хор хөнөөлгүй юу?" гэж нягтлах. |
| **5. Цагийн менежмент** | Бодит болон онлайн харилцааны тэнцвэрийг хадгалж, харилцаанд цаг хугацааны соёлыг баримтлах. |

---

## THINK зарчим

Дижитал орчинд аливаа сэтгэгдэл бичих, мэдээлэл хуваалцахаасаа өмнө **THINK** зарчмыг хэрэгжүүлцгээе:

> ### **T** - Is it **True**? (Энэ үнэн үү?)
> ### **H** - Is it **Helpful**? (Энэ тустай юу?)
> ### **I** - Is it **Inspiring**? (Энэ урам зориг өгөх үү?)
> ### **N** - Is it **Necessary**? (Энэ зайлшгүй хэрэгтэй юу?)
> ### **K** - Is it **Kind**? (Энэ эелдэг үү?)
{: .important }

## ✅ Өөрийгөө шалгах
*Та дижитал харилцаанд оролцохдоо дараах зүйлсийг анхаардаг уу?*

- [ ] Би албан и-мэйл, чатаар харилцахдаа мэндчилгээ болон ёс зүйн хэм хэмжээг баримталдаг.
- [ ] Би баталгаагүй эх сурвалжийн мэдээллийг бусдад түгээхээс татгалздаг.
- [ ] Би бусдын бүтээлийг ашиглахдаа заавал эх сурвалж зааж, оюуны өмчийг хүндэлдэг.
- [ ] Би бусдын нэр хүндэд халдсан сөрөг сэтгэгдэл, үйлдэл гаргадаггүй.
- [ ] Би бодит амьдрал болон онлайн харилцааны тэнцвэрийг хадгалж, оройн цагаар албан чат бичихээс зайлсхийдэг.
- [ ] Би сургуулийнхаа дотоод мэдээлэл, нууцлалыг чандлан хадгалдаг.
